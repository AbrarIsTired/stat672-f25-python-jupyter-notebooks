# Python & Database Management Portfolio

Portfolio of assignments completed for **STAT 672: Python & Database Management**

## Overview
This repository contains Jupyter notebooks for various topics in Python:
- Python fundamentals (variables, loops, functions, debugging)
- Data structures (lists, strings, dictionaries)
- NumPy and Pandas for data manipulation
- Data visualization and exploratory analysis
- Object-oriented programming (classes and modules)
- Statistical analysis and simulation

## Structure
- **assignments/**: Individual module assignments covering core Python concepts and data analysis

## Technologies
- Python 3.13
- Jupyter Notebooks
- NumPy, Pandas
- Matplotlib/Seaborn (for visualization)
